Installation Instructions

  1. Unzip all files to your local site, the folder location on your computer that holds your website files.
  2. *Upload all files and folders to your web server exactly as they were unzipped (DO NOT MODIFY file or folder names)
  3. Start using your application! - you can access the display page and admin page through the following url's (you need to substitute your domain name):
  • Location of start page: http://www.yourdomainname.com/linksmanager/
  • Your default login credentials for the Admin Area are:

    • username: admin
    password: admin

  • Visit the Product Support page to access the user manual, plugin wizard and supporting documentation

*Important Note If Upgrading from version v1 - Simply replace the v2 ms access database with your v1 ms access database.

Modifying Your Admin Security Credentials

  1. Open "http://www.yourdomain.com/linksmanager/admin_login.asp"
  2. Go to line 13 to set your login username. Please note that your username has to be within the quotes.
    example:
    Username = "myusername"
  3. Go to line 16 to set your login password. Plase note that your password has to be within the quotes.
    example:
    Password = "password"
  4. Go to line 20 to set your admin security email. Please note that your email has to be within the quotes.
    example:
    AdminEmailAddress = "myname@domain.com"
  5. Save this file and upload it to your web server.

Configuring "Lost Password" Feature

  1. Open "http://www.yourdomain.com/linksmanager/admin_login.asp"
  2. Go to line 32 to set your mail server configuration.
    - choose 1 if your webhosting supports CDOSYS mail server
    - choose 2 if your webhosting supports CDONTS mail server - (CDONTS is default setting)
  3. If you select 1 (CDOSYS), you will need to do this steps:
    - Go to line 38 to set your SMTP Server address. Example: EmailServer = "mail.yourdomain.com"
    - Go to line 44 and set the value to 1 if your hosting requires SMTP Authentication. Also, go to line 47 and 48 to set your authentication username and password.
  4. Save this file and upload it to your web server.

Note: Please contact your web host to get the information if you don't know the email configuration.

Getting Started 

We’ve all visited pages before where their “convenient” link page is more like a link graveyard. Not only does this waste some of the precious time we’ve invested into that website, we also begin to suspect that the information on the whole website is faulty. Chances are, we’ll move along.

DMXReady’s Links Manager makes the process of keeping your link page updated easy. Rather than change and upload an entire page, your clients can quickly update each link in real time on their Admin page, accessed through any standard browser. As soon as a dead or moved link is discovered, it can be removed or changed. And adding new links is a snap too.

Combine with DMXReady’s Contact Us Manager, and your client’s visitors will be able to send dead link information immediately for even faster updating.

Your client wants a professional site – that’s why they came to you. Don’t risk tarnishing that image with dead links!

DMXReady Designer's Tip: Determine the chain of links...

For those with a large number of links, you may want to do a bit of planning ahead of time. Break down the links into separate categories, or at least identify what those categories will be. Although DMXReady's Links Manager is easy to use, pre-planning will make using this tool even easier!

Creating A Links Page with DMXReady Links Manager 

Add New Category 

When you first install DMXReady Links Manager into your website, you will note a sample database that you may experiment with. Once you are ready to start your own Links page, you can delete these entries as shown below.

  1. Login to your admin area through by clicking on the admin login link: http://www.yourdomain.com/linksmanager/ Once logged in, you will see yyour admin actions (add, edit, delete).
  2. Click on Insert New.
  3. Click on Add/Edit New Category.
  4. A pop-up box will appear. In the text box next to Add New Category, type the name of your first category and click on Insert New.
  5. The pop-up window disappears and you are back to the Links creation page. Repeat the above process until all your categories are included.

Edit Category 

Once you have designated a category, it can be easily changed, replaced, or deleted. This can be done at anytime during creation or administration of the website; all changes made to category names will be automatically updated throughout the Links.

  1. In the Links creation page, click on Add/Edit New Category.
  2. When the pop-up window appears, select the category you wish to change. Once you've made the changes, click on Update.
  3. To delete the category altogether, click on Delete.

Insert New Link 

Once you have created your categories, you are ready to add all current Links. (Don’t worry if you think of another category during this process, just click on Add/Edit Category again and you can immediately insert it.)

  1. On the Insert New Item page, use the pull-down menu to select the category you want to enter the document under (if applicable).
  2. In the corresponding text boxes, type the link name, description, and URL.
  3. Click Publish to Links Listing Page. The event information will automatically be uploaded to your server.
  4. Repeat this process until all current events are inserted into your Links.

Editing a Link 

  1. To change a link name, description, or URL, go to the Show All page (the first page that appears when you enter the admin page).
  2. Click Edit next to the Links that you wish to change. All the document's information will appear just as you have entered it. Make the required changes and click Publish to Links Listing Page.

Deleting a Link 

  1. To delete a Link, go to the Show All page (the first page that appears when you enter the admin page).
  2. Click Delete next to the Links that you wish to remove. All information will be immediately removed from the server.
DMXReady Designer's Tip: To Avoid Losing Links Information Altogether...

If it is at all possible that you may want to reuse the Link again in the future, but you do not want that particular Link on the website at the moment, you may find it better to simply deactivate the Link by clicking off the 'Activate' checkbox in the editing page. The Link will no longer appear to visitors of your page, but the information will still be available for you to change or activate as you need in the future.
